# 基本环境搭建

1. 在宝塔面板上安装 Apache ，mysql , php
2. 设置 mysql 远程连接
3. 使用 navicat 远程连接数据库，创建 wordpress 数据库，默认字符集 utf-8

 

4. 创建网站，设置域名。  
5. 打开网站根目录上传 wordpress 中文版压缩包，然后解压。  
6. 将解压出来的 wordpress 文件夹里的使用文件剪切到网站根目录。  
7. 在网站根目录创建 wp-config.php 文件。
