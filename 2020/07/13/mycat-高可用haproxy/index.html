# haproxy 作用

HAProxy 厉害的点，不需要写 mysql 的健康检测，软件本身可以检测出 mysql 服务停止，然后切换到正常的服务器上，而且当 mysql 服务器重启之后，HAProxy 不需要重启，也能识别并且切换上去。
