  3. start 文件名 打开文件
  4. type 文件名 查看文件
  1. copy con 文件名 创建文件 并写入内容 ctrl+Z 停止输入 enter 保存
  2. md 目录名 创建目录
  3. rd 目录名 删除目录
  4. ren 原名 新名 修改文件夹或文件名称
  5. del 文件名 删除文件
  6. copy 文件名 目标路径 拷贝文件
  7. move 文件名 目标路径 移动文件
  8. tree 列出文件树
  9. net use k: \ip\c$ 将远程主机 c 盘映射到本地主机 K 盘
  10. net use k: \ip\c$ del 删除
  11. net start 查看开启的服务
  12. net start 服务名 开启服务
  13. net stop 服务名 关闭服务
  14. net share 查看共享
  15. net share ipc$ 创建共享
  16. net share ipc$ /del 删除共享
  17. netstat -a 查看端口
  18. netstat -n 查看端口网络状态
  19. netstat -v 查看正在进行的工作
  20. net user 用户名 密码 /add 添加用户
  21. net user 用户名 /del 删除用户
  22. net localgroup administrator 用户名 /add 添加用户到 administrator 组
  23. calc 打开计算器
  24. charmap：启动字符映射表
  25. 自动关机命令
  • Shutdown -s -t 600：表示 600 秒后自动关机
  • shutdown -a ：可取消定时关机
  • Shutdown -r -t 600：表示 600 秒后自动重启

  3. tasklist | findstr pid 根据 pid 查询程序
  4. taskkill /f/t /im 软件名称 杀死程序
