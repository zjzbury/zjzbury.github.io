# 名词解释

  • 人力资源规划

人力资源规划（Human Resource Plan, 简称 HRP) 也叫 “人力资源计划”，是指为实施企业的发展战略，完成企业的生产经营目标，根据企业内外部环境和条件的变化，通过对企业未来的人力资源的需求和供给状况的分析及估计，运用科学的方法进行组织设计，对人力资源的获取、配置、使用、保护等各个环节进行职能性策划，制订企业人力资源供需平衡计划，以确保企业在需要的时间和需要的岗位上，获得各种必需的人力资源，保证事（岗位）得其人、人尽其才，从而实现人力资源与其他资源的合理配置，有效激励、开发员工的规划。

  • 员工招聘

员工招聘是指根据企业的人力资源规划所确定的人员需求，通过多种渠道，利用多种手段，广泛吸引具备相应资格的人员到企业任职和工作的过程

  • 绩效管理

是指企业的各层级管理者和员工为达到企业的目标，共同参与绩效计划的制订、绩效辅导沟通、绩效考核评价、绩效结果应用、绩效目标提升的持续循环过程。缋效管理的目的是持续提升个人、部门和企业的绩效。绩效管理的过程通常被看作一个循环，这个循环分为四个环节，即： 绩效计划、绩效辅导、绩效考核和绩效反馈。 在绩效管理的四个环节中，绩效计划的制订是绩效管理的 基础环节 ， 绩效辅导沟通是绩效管理的 重要环节 ，绩效考核评价是绩效管理的 核心环节 。

  • 薪酬管理

是指企业针对所有员工所提提供的服务来确定他们应当得到的报酬总额以及报酬结构和报酬形式的一个过程。在这一过程中，企业就薪酬水平、薪酬体系、薪酬结构以及特殊员工群体的薪酬做出的决策。企业的薪酬管理系统一般要同时达到公平性、有效性和合法性三大目标。

  • 员工培训

员工培训是指企业为开展业务及培养人才的需要，采用各种方式对员工进行有目的、有计划的培养和训练的一种管理活动。其中，公开课、内部培训等均为常见的员工培训及企业培训形式

# 主观题

  • 人力资源规划的程序
  1. 收集有关信息资料；
  2. 人力资源需求预测；
  3. 人力资源供给预测；
  4. 确定人力资源净需求
  5. 编制人力资源规划；
  6. 实施人力资源规划；
  7. 人力资源规划的评估；
  8. 人力资源规划的反馈与修正
  • 工作分析与人力资源规划的关系
  1. 人力资源规划是人力资源管理的重要组成部分；
  2. 人力资源规划要依赖于工作分析的结果。
  • 人力资源管理的六大模块

包括： 人力资源规划、招聘与配置、培训与开发、绩效管薪酬福利管理和员工关系管理。

  • 工作分析在人力资源规划中的具体应用
  1. 人力资源需求预测；
  2. 人力资源供给预测；
  3. 人力资源的供求平衡
  • 工作分析在员工招聘中的作用
  1. 工作分析提供了员工招聘的依据；
  2. 工作分析提供了招聘职位的描述；
  3. 工作分析提供了招聘沟通所需要的信息
  • 工作分析在员工招聘各个环节的应用
  1. 招聘信息的确定；
  2. 招聘广告的设计；
  3. 招聘方式的选择；
  4. 候选人资料的筛选；
  5. 候选人的选拔；
  6. 人员的配置
  • 工作分析在绩效管理中的具体应用
  1. 设定绩效考核指标；
  2. 确定考核指标权重；
  3. 确定绩效考核周期；
  4. 设定绩效考核标准；
  5. 确定绩效考核主体；
  6. 确定绩效考核方法；
  7. 绩效的沟通与反馈；
  8. 绩效改进
  • 工作分析在薪酬管理中的具体应用
  1. 依据工作分析建立职位体系。
  2. 依据职位体系构建薪酬制度：①薪酬模式的选择；②薪酬结构的确定；③薪酬差距的确定；④确定薪酬水平。
  3. 工作分析在薪酬管理应用中应关注的其他问题：①企业内部的分工与协作问题；②适时的动态调整问题
  • 员工培训的程序
  1. 培训需求分析：首要环节；
  2. 制订与实施培训计划；培训效果评估与转化。
  • 工作分析在员工培训中的具体应用
  1. 培训需求分析：首要环节；
  2. 制订培训计划；
  3. 培训工作的展开；
  4. 培训成果转化
