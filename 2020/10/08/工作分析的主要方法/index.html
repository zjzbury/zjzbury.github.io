# 名词解释

  • 访谈法

访谈法是访问工作任职者，了解他们的工作内容，了解为什么这样做和怎样做，由此获得工作分析的资料

  • 问卷调查法

以书面的形式，通过工作任职者或其他相关人员单方面信息传递来实现的工作信息收集方式。

  • 资料分析法 1810

资料分析法是一种普遍通用的方法，它是通过对现有资料的收集，借助资料分析工具，直接提炼有用的工作分析信息的一种方法。

  • 观察法

观察法又名观察分析法，一般是由有经验的人，通过直接观察的方法，记录某一时期工作的内容、形式和方法，并在此基础上分析有关的工作因素，达到分析目的的一种方法。

  • 工作日志法

工作日志法是为了了解员工实际工作的内容、责任、权利、人际关系及工作负荷，而要求每个员工都要将自己所从事的每一项活动按照时间顺序以日志的形式进行记录，以实现工作分析目的的一种工作分析法。

# 主观题

  • 简述访谈法运用的原则
  1. 与主管人员密切合作；
  2. 设计一份具有指导性的问卷或提纲；
  3. 注意访谈地点的选择，创造良好的访谈环境；
    4 . 必须尽快与被访谈者建立起融洽的关系；
  4. 访谈时尽力避免谈论 “人”；
  5. 在进行群体访谈时须有直接上线主管人员在场；
  6. 对没有规律的工作方式的内容进行一一列举；
  7. 在访谈完成之后，要对资料进行核查与核对
  • 试述访谈的技巧 1910
  1. 事先清晰地说明访谈的目标和方法。
  2. 在访谈前，确认访谈是不是得到所要信息的最合适工具。
  3. 选择适当的回答者以满足所寻求信息的性质、资料收集的方式和研究的其它要求。
  4. 为达到访谈的目的需要取得回答者的支持。
  5. 控制访谈，使访谈指向一定的目标。
  6. 控制个人举止、行为等其他会影响结果的因素。
  7. 记下意外的重要信息，尤其是正式访谈计划中没有想到的或新的信息。
  • 简述工作分析中访谈法的优缺点
优点 缺点
易于控制多方面的信息 一旦被访谈者对访谈的动机持怀疑态度，则回答问题时就会有所保留，或工作访谈者技巧不佳等因素而造成信息扭曲
可获得完全的工作资料以免去员工填写工作说明的麻烦 分析项目繁杂时，费时又费钱
可以进一步使员工和管理者沟通观念，以获取谅解和信任 工作分析者的观点影响到对工作信息正确的判断。
可以不拘形式，问题内容较有弹性，又可随时补充和反问，这是填写问卷办不到的。 占用员工工作时间，妨碍生产
收集方式简单，易于操作，适用于对文字理解有困难的人 面谈者易从自身利益考虑而导致信息失真
可以使被访谈者更了解工作分析的目的和必要性，同时对工作进行系统性的思考、总结和提炼。
  • 简述问卷调查法的操作实施步骤。
  1. 问卷设计；
  2. 问卷试测；
  3. 样本选择；
  4. 问卷发放及回收；
  5. 问卷处理及运用
  • 简述问卷调查法的优缺点。
优点 缺点
利用问卷调查表采集信息，采集的信息准确、全面 问卷设计质量不高时，会严重影响分析的结果
信息采集快速、简便、经济。 员工缺乏表达能力的情况下，这种方法也不理想
员工比较容易接受 某些员工出于种种原因在填写问卷时可能会夸大其词，这时所得到的答案不能真实地反映实际情况
它弥补了工作分析者水平不一的弱点 员工对工作分析技术不太清楚，在填写问卷调查表时不知如何填写，往往需要多次反复填写与修订才能达到要求
通过员工自己填写本岗位的工作分析调查表，可对本岗位的工作职责、工作权限、工作流程、任职资格等进行一次梳理，进一步增强对本岗位工作的认识和责任感。
问卷调查法的信度和效度比较高。
  • 资料分析法的操作步骤
  1. 确定工作分析对象；
  2. 确定信息来源；
  3. 收集各方面得到的可利用的原始资料；
  4. 筛选整理相关信息；
  5. 描述信息
  • 简述资料分析法的优缺点。
优点 缺点
成本较低。 缺乏灵活性
工作效率较高 — 般收集到的信息不够全面，尤其是小型企业或管理落后的企业，往往 无法收集到有效、及时的信息
能为进一步工作分析提供基础资料和信息
  • 简述工作分析中观察法的应用要求和适用范围。
# 1. 观察法应用的要求

（1）注意所观察的工作应具有代表性。
（2）观察人员在观察时尽量不要引起被观察者的注意。
（3）观察前应确定观察计划。观察计划中应含有观察提纲、观察内容、观 察时刻、观察位置等。
（4）观察时思考的问题应结构简单，并反映工作有关内容，避免机械记录。
（5）在使用观察法时，应用适当的方式将工作分析人员介绍给员工，使之能够被员工接受。
(6) 注意所观察的工作相对稳定，即在一定时间内，内容、程序以及对工作人员的要求都不会发生明显变化的工作
（7）注意观察细微之处，观察者进入观察现场，不能只看到大的方面，还要注意防止遗漏那些关键的细微的工作要求

  • 2. 观察法的适用范围

观察法适用于短时期的外显行为特征的分析，适合于比较简单、不断重复，又容易观察的工作分析，这些工作内容主要是由身体活动来完成的工作，而不适用于隐蔽的心理素质的、没有时间规律与表现规律的，即脑力劳动成分比较高的工作和处理紧急情况的间歇性工作。有些工作内容中包括许多思想和心理活动、创造性和运用分析能力，这些工作就不宜使用直接观察法

  • 观察法的操作步骤。
  1. 观察的准备阶段；
  2. 进行观察；
  3. 进行面谈；
  4. 合并工作信息；
  5. 核实工作描述。
  • 简述观察法的优缺点。
优点 缺点
全面性 干扰工作正常行为或工作者的心智活动。
手段多样，效率较高 无法感受或观察到特殊事故，如果工作本质上偏重心理活动，则观察的成效有限。
要求观察者有足够的实际操作经验。
不能得到有关任职者资格要求的信息。
不适用于工作循环周期长、脑力劳动等工作。
  • 简述工作日志法的操作流程。
  1. 准备阶段；
  2. 工作日志的填写阶段；
  3. 信息的分析整理阶段。
  • 使用工作日志法的注意事项有哪些。
  1. 核心关键岗位，其职责重大或稳定性差，则不宜用工作日志法。
  2. 工作日志所获得的信息十分繁杂，后期信息整理工作量极大。需设计结构化程度较高的填写表格。
  3. 应加强与填写者的沟通交流，削弱信息交流的单向性。
  4. 为任职者提供帮助与支持，力图减少填写偏差。
  • 简述工作日志法的优缺点。
优点 缺点
信息可靠性很高； 将注意力集中于活动过程，而不是结果，且容易干扰员工工作。
适于确定有关工作职责、工作内容、工作关系、劳动强度方面的信息； 使用这种方法必须做到，从事这一工作的人最此项工作的情况与要求最清楚。
所需费用较少，对分析高水平与复杂的工作，显得比较经济、有效。 整理信息的工作量大，归纳工作繁琐。
适用于管理或其他随意性较大，内容复杂的岗位分析。 员工可能会夸大或隐藏某些活动，填写者因不认真可能会漏填某些内容，可能带有主观色彩，从而影响到工作分析结果
  • 关键事件法的操作步骤
  1. 识别岗位关键事件；
  2. 记录关键事件信息和资料；
  3. 对关键信息资料进行分
  • 简述职位分析问卷法的优缺点。
优点 缺点
PAQ 法可以直接用于不同的组织、不同的工作，使得比较各组织见的工作更加容易，也使得工作分析更加准确与合理。 通用化或标准化的格式导致了工作特征的抽象化，所以不能描述实际工作中特定的、具体的任务活动
PAQ 得出的每一种（或每一类）工作的技能数值与等级还可以用来进行工作评估和人员甄选。 由于保证收集数据的有效性、准确性和通用性的要求，需要很高的时间成本。同时需要问卷的填写人受过专业训练的工作人员，由他们对任职者和直接主管进行工作内容的访谈，然后再填写 PAQ 问卷
同时考虑员工和工作两个变量因素，并将各种工作所需的基础技能与基础行为以标准化的方式罗列出来，为人事调查、薪酬标准制定提供了依据。
  • 简述职位分析问卷法的实施步骤。（可不背）
  1. 明确工作分析的目的。
  2. 赢得组织的支持。
  3. 确定信息收集范围与方式。
  4. 培训 PAQ 分析人员。
  5. 与员工沟通整个项目。
  6. 收集信息并编码。
  7. 分析工作分析结果。
  • 简述管理职位描述问卷法的优缺点。（可不背）
优点 缺点
MPDQ 是用于评价管理工作的职位分析工具，适用于管理层的职位分析，有很强的针对性。 MPDQ 的各个分析维度是在对国外管理人员的实证研究基础上形成的，缺乏根据中国管理人员自身特点的修正。
经过职位分析专家长期的广泛深入的实证研究和修订，MPDQ 具有较高的区分度，并能将数据信息转化为人力资源管理人员可以使用的信息报告和表格。 由于管理工作的复杂性，难以用 MPDQ 分析所有类型的管理工作。
由于 MPDQ 收集信息的广泛性，使其可以在人力资源管理的其他职能领域进行综合应用。 成本比较高，投入比较大。
通过计算机程序，MPDQ 在某种程度上降低了主观因素的影响，同时其最终报告大量以图表形式出现，信息充足，简单易懂，提高了组织人力资源管理的效率。
  • 简述职能工作分析法的框架。（可不背）

职能工作分析法的框架包括以下几个方面：
（1）完成什么与做什么。
（2）工作者的职能 —— 事、数据、人。
（3）完整意义上的工作者。
（4）工作系统。

  • 简述在选择工作分析方法时，企业需要考虑的因素。
  1. 工作分析的目的。
  2. 成本。
  3. 工作性质。
  4. 待分析的工作样本数量。
  5. 分析对象。
