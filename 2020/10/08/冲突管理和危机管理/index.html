# 名词解释

# 角色矛盾

指企业中由于职位的不同而存在多种决策，如销售人员、财务人员等，他们在不同的岗位上，对待同一个问题或现象时，思考的角度可能不太一样，因此就容易产生矛盾

# 环境应激

环境应激是指当竞争激烈形成工作压力时，人们会对环境作出不同程度的反应，这时候彼此之间就可能产生冲突

# 员工关系危机 1901

员工关系危机是指员工与员工之间、员工与企业之间因各自的利益和发展取向的不同，所引起或可能引起的矛盾冲突

# 主观题

# 避免破坏性冲突应坚持的原则及其对策
  • （一）避免破坏性冲突发生应坚持原则： （以理充公）
  1. 理性适当原则。
  2. 充分了解原则。
  3. 公平原则。
  4. 以人为本的原则。
  • （二）避免破坏性冲突发生的对策： （预测；沟通；优化）
  1. 建立冲突预测机制。
  2. 积极沟通。
  3. 优化组织架构。
# 简述冲突预防的具体措施 （制定制度防止冲突；加强沟通减少隔阂；增强“大集合”意识；完善沟通机制）
  1. 制定完善的规章制度，以防止制度疏漏、职责不清等因素引起的消极性冲突。
  2. 加强团队之间、个人之间以及个人与团队之间的信息和意见的沟通，以减少隔阂和分歧。
  3. 增强企业员工的集体意识、合作意识和大局意识。
  4. 完善企业内部沟通、信息系统和机制
# 管理者解决员工冲突的沟通技巧 倾听心声；注意用词；尊重理解；重述明白；解决方法；关注改变。
  1. 倾听处在冲突中的员工的心声。
  2. 要员工注意词汇的运用。
  3. 管理者带着尊敬和同理心倾听员工的看法，显示自己的理解。
  4. 要求每个员工重述他们刚才听到的对方的观点，以确保每个人真正明白对方说的是什么。
  5. 寻找解决问题的办法。
  6. 关注行为的改变。
# 试述冲突的含义、产生原因及处理策略
  • （一）含义：

企业组织中的成员、群体、组织在交往中产生意见分歧，出现争论、对抗，导致彼此间关系紧张，该状态称为冲突。

  • （二）冲突产生的原因。
  1. 员工的 “个性” 差异；
  2. 信息缺乏；
  3. 角色矛盾；
  4. 环境应激；
  5. 沟通障碍；
  6. 企业资源的争夺；
  7. 岗位职责规定不明确；
  8. 不合理的考核制度；
  9. 权利的争夺和风气不正等；
  10. 价值观和信仰的差异。
  • （三）处理冲突的策略
  1. 合作策略。
  2. 分享策略。
  3. 回避策略。
  4. 竞争策略。
  5. 第三者策略。
  6. 调和策略。
# 解决员工冲突的步骤及注意问题
  • （一）步骤：
  1. 界定冲突；
  2. 分析冲突；
  3. 鼓励双方看到积极的方面；
  4. 整合期望值；
  5. 确定和解目标；
  6. 建立解决冲突的框架；
  7. 计算冲突成本。
  8. 明确义务；
  9. 沟通总结。
  • （二）应注意的问题： 弄清原因；公平对待；对症下药。
  1. 要尽量弄清下属间矛盾冲突产生的原因是什么，矛盾发生的过程、程度以及影响范围有多大。
  2. 无论处理什么样的矛盾，管理者对当事双方一定公正对待。
  3. 管理者要针对不同的冲突内容与程度选择相应的解决办法
# 试述员工关系危机的含义及其产生的原因 2001
  1. 错误的行为。
  2. 利益异议。
  3. 不公平或不合理事件。
  4. 关系的不协调。
  5. 相同需求员工的组合。
  6. 信息的障碍。
# 管理者在解决员工关系危机时，必须掌握的技巧
  1. 确认员工关系危机的阶段。
  2. 找到团体中的核心人物。
  3. 进行宣传。
  4. 个别应对。
  5. 争取支持。
  6. 充分考虑员工利益。
  7. 进行谈判、说服
